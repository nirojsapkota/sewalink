    It should request `navigator.mediaDevices.getUserMedia({ audio: true })`.
    Manage `MediaRecorder` instance to start/stop recording.

    Update UI state (e.g., 'Recording...', 'Processing...', idle) via CSS class toggling or text changes on targets.
  </action>
