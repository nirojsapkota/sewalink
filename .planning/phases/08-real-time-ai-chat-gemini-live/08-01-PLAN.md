<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → ActionCable | Unencrypted audio chunks cross into the server. Protected by HTTPS/WSS. |
| Rails → Gemini API | API keys and user voice data sent to external provider. Protected by GEMINI_API_KEY and TLS. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01-01 | Information Disclosure | GEMINI_API_KEY | mitigate | Store in ENV or Rails Credentials; never expose to frontend. |
| T-08-01-02 | Denial of Service | ActionCable Channel | mitigate | Limit audio chunk frequency (max 10 chunks per second) in Stimulus. |
| T-08-01-03 | Elevation of Privilege | Tool Execution | mitigate | Validate all tool arguments (title length, budget range) before applying to database. |
| T-08-01-04 | Tampering | Audio Data | mitigate | Enforce WSS (Secure WebSockets) for all browser-server communication. |
</threat_model>
